学习笔记
寻路问题
1.本质：
  1）根据一个起点，找到该点周围可以到达的点（九宫格的中心）用数组存储
  2）根据下一个点再次寻找其周围可达的点（障碍物或已添加则跳过），直至找到终点
2.疑问：
  1）如何标记路线：
  在当前点记录上一点（到达该点）的坐标，找到重点后由后向前一次找到起点
  2）如何找到最短路程：
  从数组里取下一个计算的点时取出距离（distance）终点最小的点
3.算法与数据结构：
  队列：先进先出；栈：后进先出
  定义一个数据结构（class），设定其存与取的规则及其他属性（length等）
  算法：A*
